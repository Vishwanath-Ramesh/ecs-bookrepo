## Summary

This is a boilerplate setup project for React application

## Tools & Libraries used

- React
- JavaScript
- Axios
- Webpack
- Babel

## Project Setup

Install dependencies and devdependencies, run:

using npm

```
npm install
```

using yarn

```
yarn install
```

## Running the app

To get the app up and running, run:

```shell
npm run dev
or
yarn dev
```
